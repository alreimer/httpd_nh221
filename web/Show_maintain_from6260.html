  Firmware upgrade
 
    Please ensure that the version to be upgraded is appropriate.
Please wait and ensure a safe operating environment.

Note: when rebooting is complete, the system will return to the "Home" page.
 
 
System restart
 
   
This may take several minutes.
Please wait and ensure a safe operating environment.

Note: the system will return to the "Home" page after rebooting.
 
  Reset to factory default
 
    It may take several minutes. Current configuration will be lost.
Please wait and ensure a safe operating environment.

Note: when the reset is complete, the system will return to the "Home" page.
